# Описание будущего процесса разработки

## ML-component

Данная часть будет иметь следующую структуру:

```shell
|
├── data/
|   └── image.jpg               # Ваше тестовое изображение
|
├── admm_core.py                # Ядро алгоритма ADMM
├── utils.py                    # Вспомогательные функции (загрузка данных, маски)
├── visualize.py                # Функции для отрисовки результатов
└── main_model.py               # Главный скрипт для запуска всего процесса
```

Описание файлов:
* `admm_core.py`: Содержит только логику самого алгоритма.
* `utils.py`: Функции, которые могут понадобиться в разных частях проекта: загрузка и предобработка изображения, создание маски повреждений.
* `visualize.py`: Весь код, связанный с matplotlib, будет вынесен сюда.
* `main_model.py`: Точка входа. Он управляет всем процессом: задает параметры, вызывает функции из других модулей и показывает результат.
