# DenoiseBuddy: Телеграм-бот для обработки изображений

## Описание проекта

DenoiseBuddy — это телеграм-бот, который помогает пользователям обрабатывать изображения путем удаления шума и улучшения их качества. Бот также предоставляет возможность генерации новых изображений с помощью нейронных сетей.

## Архитектура проекта

Проект состоит из трех основных компонентов:
1. Бэкенд-часть (серверная логика и база данных)
2. Фронтенд-часть (интерфейс телеграм-бота)
3. ML-компонент (модели машинного обучения)

### Бэкенд-часть

#### База данных

База данных для хранения информации о пользователях и их активности:

**Таблица пользователей:**
- ID пользователя
- Логин/имя пользователя
- Пароль (учебный проект)
- Тариф/оставшиеся токены (опционально)

**Таблица файлов:**
- ID файла
- ID пользователя
- Путь к исходному изображению
- Путь к обработанному изображению
- Время загрузки

**Таблица запросов:**
- ID запроса
- ID пользователя
- Содержание запроса
- Время запроса

#### Функциональность бэкенда

1. **Управление пользователями**
   - Простая регистрация и вход пользователей
   - Базовый учет токенов (опционально)

2. **Обработка изображений**
   - Прием и обработка загруженных изображений
   - Взаимодействие с ML-моделями для удаления шума

3. **Генерация изображений**
   - Обработка текстовых запросов на генерацию
   - Интеграция с доступным API нейросетей

4. **Управление историей**
   - Хранение истории запросов пользователя
   - Возможность очистки истории

### Фронтенд-часть (Телеграм-бот)

#### Пользовательский интерфейс

1. **Система навигации**
   - Меню с кнопками для основных функций
   - Инлайн-кнопки для удобства навигации

2. **Взаимодействие с пользователем**
   - Редактируемые сообщения (вместо отправки новых)
   - Простые индикаторы загрузки

3. **Основные функции**
   - Загрузка и обработка изображения
   - Генерация изображения по запросу
   - Просмотр истории с возможностью перехода к конкретному запросу
   - Очистка истории

### ML-компонент

#### Модели для обработки изображений

1. **Модель удаления шума**
   - Простая сверточная нейронная сеть
   - Базовое шумоподавление

#### Датасет

- Небольшой набор изображений для обучения и тестирования
- Разделение на тренировочную и тестовую выборки

## Технический стек

- **Язык программирования:** Python
- **Телеграм API:** python-telegram-bot
- **База данных:** SQLite (для простоты)
- **ML-фреймворк:** PyTorch или TensorFlow

## Дорожная карта разработки

1. **Этап 1: Базовая функциональность**
   - Создание простого телеграм-бота
   - Настройка базы данных
   - Реализация загрузки и сохранения изображений

2. **Этап 2: ML-компонент**
   - Разработка/адаптация модели удаления шума
   - Интеграция модели с ботом

3. **Этап 3: Улучшения и дополнительные функции**
   - Добавление функции генерации изображений
   - История запросов с навигацией
   - Улучшение интерфейса

## Команда разработки

- Участник 1: Разработка бота и интерфейса
- Участник 2: Бэкенд и база данных
- Участник 3: ML-модель для удаления шума
- Участник 4: Интеграция API для генерации и общая координация
